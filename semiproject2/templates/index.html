About the Machine Learning Process

k-NN

기존 데이터 중 가장 가까운 k개의 데이터를 정보로 새로운 데이터를 분류하는 방법. k값 조정을 통해 정확도가 생각보다 많이(4.33%) 상승했습니다. ACC : 52.02%

SVM

주어진 데이터가 어느 카테고리에 속할지 판단하는 이진 선형 분류 모델. 데이터셋이 많아서인지 학습 속도가 너무 느려서 적합하지 않았습니다. ACC : 51.0%

Decision Tree

root에서부터 특정 기준(질문)에 따라 데이터를 구분하며 진행하여 최종 결정을 내리게 되는 모델. ACC : 50.83%

Random Forest

여러 Decision Tree가 모여 구성된 모델. Decision Tree들이 내린 예측 값들 중 가장 많이 나온 값을 최종 예측값으로 정하기 때문에 Decision Tree보다 나은 정확도를 얻은 것을 확인할 수 있습니다. ACC : 51.25%

Logistic Regression

종속 변수가 범주형 데이터를 대상으로 하며 입력 데이터가 주어졌을 때 해당 데이터의 결과가 특정 분류로 나뉘는 모델. ACC : 51.22%

Conclusion

Size라는 범주형 데이터를 사용하였기 때문에 분류(Classification) 모델을 사용했고, 그 중에서 정확도가 가장 높고 실행시간도 비교적 빠른 k-NN 모델을 채택했습니다
